# Loop-challenges-
Printing the Odd number: in this code, I used a for loop to go through numbers 1 to 20 and used the modulus operator (%) to determine if the number is not divisible by 2. If the number is odd, it will then be printed in the console.
Decreasing multiples of three: in this code, the for loop counts down from 100 to 0 and for each number I used the modulus operator (%) to check if it is divisible by 3. If the condition is met, the number is printed in the console. 
Print the given sequence: in this code, I set the for loop to start at 4 and end at -3.5 like the given sequence and decrease it by 1.5 each time until it reaches -3.5. Thus, this sequence (4, 2.5, 1, -0.5, -2, -3.5) is printed in the console.  
Sigma: in this code, I made a variable named (sum) and set it to 0  to store the running total. I made the loop start at 1 and run until it reaches 100. Inside the loop, each number is added to sum using (sum = sum+i) making the code update the total by adding the current number at every step. After the loop finishes, all numbers from 1 to 100 will be added together and printed in the console. 
Factorial: in this code, I made a variable called product and initialized to 1 for the multiplication process that follows. The for loop starts at 1 and continues up to 12. During each iteration, the current number (i) is multiplied with (product) using (product = product *i) after the loop is finished the final result is printed in the console.
